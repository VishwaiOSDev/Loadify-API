Loadify

Description

Loadify is an iOS and iPad app designed for downloading high-quality content from YouTube, including videos and shorts, as well as Instagram reels and posts.

Base URL

The base URL for all API requests is:

https://loadify.madrasvalley.com/api

Endpoints

YouTube

  • [GET] Listing YouTube Video Details

    GET /yt/details

    Provides details about a YouTube video.

    Parameters

    ParameterTypeRequirementDescription
    urlStringRequiredThe URL of the YouTube video

    Response

    Returns a JSON object with following properties:

    • title (String): Title of the YouTube video.
    • description (String): Description of the YouTube video content.
    • lengthSeconds (Integer): Duration of the video in seconds.
    • viewCount (Integer): Number of views the video has received.
    • publishDate (String - ISO 8601 format): Date and time when the video was published.
    • ownerChannelName (String): Name of the YouTube channel that owns the video.
    • videoId (String): Unique identifier for the video.
    • author (Object): Information about the video's author.
      • id (String): Author's unique identifier.
      • name (String): Author's name.
      • user (String): Author's username.
      • channel_url (String): URL of the author's YouTube channel.
      • verified (Boolean): Verification status of the author's channel.
      • subscriber_count (Integer): Number of subscribers to the author's channel.
    • likes (Integer): Number of likes the video has received.
    • video_url (String): URL link to the YouTube video.
    • thumbnails (Array): Array of thumbnail objects.
      • url (String): URL of the thumbnail image.
      • width (Integer): Width of the thumbnail image.
      • height (Integer): Height of the thumbnail image.

TikTok

  • [GET] Listing and Download TikTok Video Details

    GET /tk/details

    Provides details about a TikTok videos.

    Parameters

    ParameterTypeRequirementDescription
    urlStringRequiredThe URL of the TikTok video

    Response

    Returns a JSON object with following properties:

    • status (String): Represents the status of the response.
    • result (Object): Contains detailed information about the TikTok content.
      • type (String): Indicates the type of content, in this case, it's "image."
      • id (String): Unique identifier for the TikTok content.
      • createTime (Integer): Timestamp indicating the creation time.
      • description (String): Descriptive hashtags related to the content.
      • hashtag (Array of Strings): Array containing individual hashtags associated with the content.
      • author (Object): Information about the TikTok content's author.
        • uid (String): Author's unique identifier.
        • username (String): Author's TikTok username.
        • nickname (String): Author's nickname.
        • signature (String): Author's signature or bio.
        • region (String): Author's region.
        • avatarThumb (Array of Strings): Thumbnails for the author's avatar in different sizes.
        • avatarMedium (Array of Strings): Medium-sized avatars for the author.
        • url (String): URL link to the author's TikTok profile.
      • statistics (Object): Various statistics related to the TikTok content.
        • playCount (Integer): Number of times the content has been played.
        • downloadCount (Integer): Number of downloads.
        • shareCount (Integer): Number of shares.
        • commentCount (Integer): Number of comments.
        • likeCount (Integer): Number of likes.
        • favoriteCount (Integer): Number of times the content has been marked as a favorite.
        • forwardCount (Integer): Number of times the content has been forwarded.
        • whatsappShareCount (Integer): Number of times the content has been shared on WhatsApp.
        • loseCount (Integer): Count related to losing.
        • loseCommentCount (Integer): Count related to losing comments.
      • images (Array of Strings): Array containing URLs of images associated with the TikTok content.
      • music (Object): Information about the music associated with the TikTok content.
        • id (Integer): Unique identifier for the music.
        • title (String): Title of the music.
        • author (String): Author or artist of the music.
        • album (String): Album associated with the music.
        • playUrl (Array of Strings): URLs for playing the music.
        • coverLarge (Array of Strings): URLs of large-sized music cover images.
        • coverMedium (Array of Strings): URLs of medium-sized music cover images.
        • coverThumb (Array of Strings): URLs of thumbnail-sized music cover images.
        • duration (Integer): Duration of the music in seconds.
